🚀 COMMENT UTILISER
═══════════════════════════════════════════════════════════════════════════════

1️⃣ IMPORTER LE COMPOSANT
   Dans ta page (ex: src/app/[locale]/event/page.tsx):
   
   import HeroCarousel from '@/components/HeroCarousel';

2️⃣ UTILISER LE COMPOSANT
   export default function EventPage() {
     return (
       <main>
         <HeroCarousel 
           title="Ton titre"
           subtitle="Ton sous-titre"
           autoplayInterval={5000}
           redOpacity={0.4}
         />
         {/* Autres sections */}
       </main>
     );
   }

3️⃣ TESTER EN LOCAL
   npm run dev
   Puis va à http://localhost:3000/ta-page

🆘 TROUBLESHOOTING RAPIDE
═══════════════════════════════════════════════════════════════════════════════

❌ Photos ne s'affichent pas
   → ls public/*.JPG (vérifie que les photos sont là)
   → Redémarre: npm run dev

❌ Filtre rouge pas visible
   → Augmente redOpacity: redOpacity={0.6}

❌ "Module not found" error
   → Redémarre le serveur: npm run dev

❌ Texte pas lisible
   → C'est normal, drop-shadow-lg ajoute une ombre
   → Ajoute plus de contraste si besoin
